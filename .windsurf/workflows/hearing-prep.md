---
name: hearing-prep
description: Prepare for a court hearing with dual-format oral argument scripts and anticipate opposition
---

Steps:
1. Ask the user: which hearing/motion, date, what the other side is likely to argue.
2. Search local-rag for all documents related to this motion.
3. Search CourtListener for case law supporting our position.
4. Review the filed motion and any responses.

5. Produce THREE deliverables:

DELIVERABLE 1 — "THE CUTOFF" (2 sentences, memorize this):
- Sentence 1: The problem (what happened, what's wrong)
- Sentence 2: The ask (what the court should do)
- Write this so the user can say it from memory while being interrupted
- No legal jargon that sounds unnatural when spoken
- This is the nuclear option — if the judge says "get to the point" this is what comes out

DELIVERABLE 2 — "JESUS WEPT" FULL ARGUMENT (60-90 seconds spoken):
- Opening: "Your Honor, [one sentence: what we're asking for]."
- Point 1: [Strongest argument, one sentence, cite the rule or case]
- Point 2: [Second argument if needed, one sentence]
- Point 3: [Only if genuinely adds something new]
- Close: "We respectfully ask the Court to [specific relief]."
- Time this mentally — at ~150 words per minute, keep under 225 words
- Write for the EAR not the eye — conversational but precise
- No "respectfully submitted" filler — every word fights

DELIVERABLE 3 — ANTICIPATION SHEET:
- What will opposing counsel likely argue? (3-5 points max)
- One-sentence rebuttal for each
- What questions might the judge ask? (3-5 most likely)
- One-sentence answer for each
- Any procedural traps to watch for (wrong standard, waiver arguments, etc.)

6. Save all deliverables to 09_Oral_Arguments/prep_notes/YYYY-MM-DD_HearingTopic.md
7. Store key arguments in memory for future reference.
