# Case Overview

This folder contains your case's core reference files. Create these after running `node scripts/setup.js`:

- **case_index.md** — Case number, parties, status, key defense arguments, next hearing
- **party_info.md** — Defendant, plaintiff, opposing counsel contact details
- **strategy_notes.md** — Defense strategy, sequencing rules, decision trees
- **case_timeline.md** — Chronological case events

These files are loaded as baseline context for every AI drafting call. Keep them accurate and up to date.
